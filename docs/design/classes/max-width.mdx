# Max Width

Sometimes you want to limit the width of a container. You can do this by adding the `max-width-*` class to the container. We use a 12 column grid system, so the `max-width-12` class will set the max width to 12 columns, which is 1440px by default.

| Class | Description | Default Width |
| --- | --- | --- |
| `max-width-1` | Sets max width to 1 column | 120px |
| `max-width-2` | Sets max width to 2 columns | 240px |
| `max-width-3` | Sets max width to 3 columns | 360px |
| `max-width-4` | Sets max width to 4 columns | 480px |
| `max-width-5` | Sets max width to 5 columns | 600px |
| `max-width-6` | Sets max width to 6 columns | 720px |
| `max-width-7` | Sets max width to 7 columns | 840px |
| `max-width-8` | Sets max width to 8 columns | 960px |
| `max-width-9` | Sets max width to 9 columns | 1080px |
| `max-width-10` | Sets max width to 10 columns | 1200px |
| `max-width-11` | Sets max width to 11 columns | 1320px |
| `max-width-12` | Sets max width to 12 columns | 1440px |
