# List Styles

Control the list marker style on ordered and unordered lists.

| Class | Description |
| --- | --- |
| `.list-none` | Removes list markers entirely |
| `.list-disc` | Uses a filled circle marker (default for `<ul>`) |
| `.list-decimal` | Uses numbered markers (default for `<ol>`) |

### Usage

```html
<ul class="list-none">
  <li>No bullet point</li>
  <li>No bullet point</li>
</ul>
```
