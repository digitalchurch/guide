# Add a Video Module to your page

import ReactPlayer from 'react-player';
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<div className='player-wrapper'>
<ReactPlayer url="https://youtu.be/Fk6qLF9UtWM"
    width="100%"
    height="100%"
    controls={true}
    playing={true}
    muted={true}
     />
</div>

If you add a video module to your page, set the video type to embed, you can paste in a link for any oEmbed video service, including YouTube and Vimeo. In fact, you can even paste in a playlist url for YouTube and the entire playlist will be embedded.
