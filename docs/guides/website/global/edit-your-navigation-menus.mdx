1. To edit your menu in your Admin Dashboard, go to Design > Navigation (your.digitalchurch.app/wp-admin/nav-menus.php)
2. Select the Navigation Menu you would like to edit in the dropdown list at the top of that page.
3. Use the left sidebar to find items to add to your navigation menu.
