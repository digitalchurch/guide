Sometimes you’ll want to customize the order of people when displayed in the page builder. For example, you might want to list staff by alphabetical order or by a totally custom order. In this article, we’ll explain how to do that.

1. On the page you’d like to customize, launch the page builder.How to launch the page builder using the page builder button in the top left corner.
2. Click on the settings icon for the module you want to customize.Click on the settings icon for the module you want to customize.
3. Select the value you would like to order the posts by. If you select Meta Value, you can order by the first or last name using the correct Meta Key. Those meta keys are person_firstname and person_lastname.Set the orderby value for the post module
4. If you’d like to do a totally custom order, use the Selection Order option in the Order By field and in the Filter section, you can use People: Match these people… and enter the names of the people you’d like to include in the order you want them included.
