# Shortcodes

We’ve custom coded several shortcodes that you can use around your site as you customize it. Shortcodes are simple text codes in square brackets that can display useful dynamic data. For instance, we use a shortcode `[primary-site]` to display your main site URL.

### `[primary-site]`
Returns the primary site url.

### `[login]`
Login Form

### `[button]`

```html
[button url="https://link.url"]Button Text[/button]
```

### `[list-fl-builder-templates]`
Lists Page Builder Templates on current site.

### `[bible]`

Displays a Bible embed.

### `[footer-bar-content]`

Displays dynamic footer bar content for links to login/admin/account/logout.

### `[dgtl_admin_menu_profile]`

This is how we display the icon in the admin menu.
