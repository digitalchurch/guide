# Announcements

## General Settings

```php
post_type = 'announcement';
```

## Single Post Meta

| Meta Key | Type/Options | Description |
---|---|---
| `announcement_cta` | string | Button text. |
| `announcement_url` | string | URL to link to. |
| `announcement_type` | select | bar, bubble, slider, or custom |
| `slider_images` | string | Array for images to display in the slider. |
| `slide_image` | string | Single image to display in the slider. |
| `slide_url` | string | URL to link to. |
| `announcement_bg_color` | string | Background color. | 
| `announcement_txt_color` | string | Text color for the announcement. |
| `announcement_class` | string | The class to display in the call to action. |
